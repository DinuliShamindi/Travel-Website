# Air Travel - Travel Agency Website

A modern and responsive travel agency website showcasing featured destinations, travel galleries, user reviews, and booking promotions.
This project is a front-end web development project built using **HTML, CSS, and Bootstrap**.

---

## Features

- **Responsive Header** with logo, login button, and search functionality
- **Featured Destinations** with images, ratings, price, and details
- **Travelling Gallery** showcasing popular destinations
- **Banner Section** with promotional offers and booking button
- **User Reviews** section highlighting customer feedback
- **Footer Section** with contact info, features, social media links, and legal links
- Fully responsive design for desktop, tablet, and mobile devices
- Hover effects and visual animations for images and buttons

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons
- Google Fonts (optional, if used)
- Responsive design principles
